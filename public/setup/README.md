# Screenshot guida setup Vercel

Metti qui i PNG mostrati nella guida di configurazione (`/admin/setup` su
Vercel — componente `components/admin/VercelSetupGuide.tsx`).

Finché un file non è presente, al suo posto compare un segnaposto che ne indica
il percorso. I nomi devono corrispondere esattamente:

| File | Cosa mostrare |
|------|----------------|
| `1-environment-variables.png` | Vercel → Project → **Settings → Environment Variables** |
| `2-add-variable.png` | Il form **Add Environment Variable** (nome, valore, selezione ambienti, Save) |

Suggerimenti:
- Larghezza consigliata ~1200px, PNG.
- Ritaglia sull'area utile; le immagini vengono mostrate a piena larghezza del riquadro.
