# Contributing to Perla

Thanks for your interest! Perla is a real-time location-sharing app built with Next.js.

## Getting Started

1. Fork the repository
2. Create a branch: `git checkout -b feat/my-feature`
3. Install dependencies: `npm ci`
4. Follow the [Setup Guide](docs/wiki/Getting-Started.md)
5. Make your changes
6. Run checks: `npm run lint && npm run build`

## Commits

We follow [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `refactor:` code change without fix or feature
- `ci:` CI/CD changes
- `i18n:` translations

## PR Process

1. Push to your fork and open a PR against `Dev`
2. Ensure CI passes (lint, build, CodeQL)
3. At least one review required before merge

## Questions?

Open a [discussion](https://github.com/NetsukiiDev/Perla/discussions).
